# Lew UI

一个用于 Vue3 的组件库。文档地址：https://lew.kamtao.com

[英文](./README.md) | 中文

## 安装

```bash
npm install lew-ui
```

## 如何使用

main.ts

```js
// main.ts
import 'lew-ui/style'
```

```vue
<script setup lang="ts">
// page.vue
import { LewButton } from 'lew-ui'
</script>

<template>
  <lew-button text="提交" />
</template>
```
