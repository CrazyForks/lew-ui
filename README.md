# Lew-UI

A Component Library for Vue3.js. Doc address: https://lew.kamtao.com

## Install

```bash
npm install lew-ui --save
```

## How to use

引入 css

```js
// main.ts
import 'lew-ui/style';
```

页面

```vue
<script setup lang="ts">
    import { LewButton } from 'lew-ui';
</script>

<template>
    <lew-button text="发送" />
</template>
```
