# Lew UI

A Component Library for Vue3. Doc address: https://lew.kamtao.com

English | [中文](./README.zh-CN.md)

## Install

```bash
npm install lew-ui
```

## How to use

```js
// main.ts
import 'lew-ui/style'
```

```vue
<script setup lang="ts">
// page.vue
import { LewButton } from 'lew-ui'
</script>

<template>
  <lew-button text="Submit" />
</template>
```
